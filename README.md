# InventarioCiclico.API

API REST desenvolvida em **ASP.NET Core** para gerenciamento de **Inventário Cíclico**, criada com foco em **boas práticas de desenvolvimento, organização em camadas e escalabilidade**.

Este projeto faz parte do meu **portfólio profissional** e está sendo desenvolvido com o objetivo de demonstrar domínio em **arquitetura de APIs, separação de responsabilidades e padrões modernos em .NET**.  
Futuramente, o projeto contará com uma aplicação **Front-end** integrada.

---

## Tecnologias Utilizadas

- **.NET / ASP.NET Core**
- **C#**
- **Entity Framework Core**
- **Arquitetura em camadas**
- **Injeção de Dependência**
- **Logging estruturado**
- **Padrão Repository e Service**
- **DTOs para comunicação entre camadas**

---

## Arquitetura do Projeto

O projeto foi estruturado seguindo uma separação clara de responsabilidades, facilitando a manutenção e a evolução da aplicação.

InventarioCiclico.API
│
├── Controllers → Endpoints da API
├── Services → Regras de negócio
├── Repositories → Acesso a dados
├── Models → Entidades de domínio
├── DTOs → Contratos de entrada e saída
└── Logs → Modelos e serviços de logging (A ser incluido)
